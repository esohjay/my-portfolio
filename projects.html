projects

gblog

This is a web application i built for a client who has passion for blogging. It is a full stack web application.

Features

  • The app allows user registration
  • Registered users have limited activities, i.e a user cannot delete or edit posts or comments that is not created by them
  • Only an admin can create a post
  • An admin can delete any comment or post
  • Only registered users can make comments, unregistered users can only read posts
  • User password can be reset
  • Admin can upload pictures from his/her device while creating posts
  • Fuzzy search function
  • Pages are well paginated

Tools

Frontend:

HTML, CSS, Javascript, Boostrap.

Backend:

Nodejs, Expressjs, MongoDB

GIT, Github
